The above navigation bar is produced with the following code:

    
     <nav>
      <div class="nav">
       <h5 class="nav-logo">Brand</h5>
       <a class="nav-item" href="#">Link 1</a>
       <a class="nav-item" href="#">Link 2</a>
       <div class="nav-item">
        <a class="nav-item dropdown" data-toggle="dropdown">Dropdown</a>
        <ul>
         <div class="background-white">
          <li class="nav-item">Item 1</li>
          <li class="nav-item">Item 2</li>
          <li class="nav-item">Item 3</li>
         </div>
        </ul>
       </div>
       <div class="nav-menu" href="#">
        <a class="nav-item" href="#">Menu Item 1</a>
       </div>
       <div class="nav-item" href="#">After menu item</div>
      </div>
     </nav>

For the dropdown menu, is necesary to load js/wing.dropdown.js at the end of the body with:

    
     <script type="text/javascript" src="js/wing.dropdown.js"></script>

Grid with col size
